## 一、路线图概述

| 阶段 | 周期 | 核心目标 |
|------|------|----------|
| **Phase 1** | 4周 | MVP版本，核心凭证管理和预警功能 |
| **Phase 2** | 4周 | 完善账户管理和绑定功能 |
| **Phase 3** | 3周 | 扩展功能（导入导出、搜索等） |
| **Phase 4** | 6周 | 云同步和多端支持 |

## 二、MVP阶段（Phase 1）

### 2.1 目标
提供完整的凭证管理和风险预警核心体验，让用户能够：
- 管理手机号/邮箱凭证
- 设置停用计划
- 查看影响范围和预警提醒

### 2.2 功能清单

| 功能 | 描述 | 状态 |
|------|------|------|
| **凭证管理** | 添加、查看、修改、删除凭证 | ✅ |
| **脱敏显示** | 凭证默认脱敏展示，查看完整值需验证 | ✅ |
| **停用计划** | 设置凭证停用时间和原因 | ✅ |
| **影响分析** | 显示凭证绑定的账户列表 | ✅ |
| **预警系统** | 凭证停用前自动触发预警 | ✅ |
| **首页看板** | 显示预警概览和快捷入口 | ✅ |
| **数据加密** | SQLCipher数据库级加密 | ✅ |

### 2.3 交付物
- 可安装的Android APK
- 核心功能测试用例
- 用户指南文档

## 三、核心功能完善阶段（Phase 2）

### 3.1 目标
完善账户管理和绑定功能，支持多账号管理场景

### 3.2 功能清单

| 功能 | 描述 | 状态 |
|------|------|------|
| **账户管理** | 添加、查看、修改、删除应用账户 | 🔄 |
| **多账号支持** | 同一应用的多个账户管理（小号） | 🔄 |
| **绑定管理** | 凭证与账户的灵活绑定 | 🔄 |
| **绑定用途** | 设置凭证在账户中的用途（登录/验证/找回等） | 🔄 |
| **账户状态** | 正常/冻结/丢失/归档状态管理 | 🔄 |
| **扩展字段** | 注册日期、地区等自定义字段 | 🔄 |

## 四、扩展功能阶段（Phase 3）

### 4.1 目标
提供数据导入导出和高级搜索功能

### 4.2 功能清单

| 功能 | 描述 | 状态 |
|------|------|------|
| **数据导出** | 支持JSON/CSV格式导出 | ⏳ |
| **数据导入** | 支持数据恢复和迁移 | ⏳ |
| **搜索功能** | 支持按关键词搜索凭证和账户 | ⏳ |
| **绑定历史** | 追溯凭证绑定关系的历史变更 | ⏳ |
| **批量操作** | 批量设置停用计划、批量解绑 | ⏳ |

## 五、云同步阶段（Phase 4）

### 5.1 目标
支持多设备数据同步和端对端加密

### 5.2 功能清单

| 功能 | 描述 | 状态 |
|------|------|------|
| **用户注册** | Hermes账号注册和登录 | ⏳ |
| **端对端加密** | 数据加密后上传，服务端无法解密 | ⏳ |
| **多设备同步** | 手机、平板、Web端数据同步 | ⏳ |
| **冲突解决** | 多设备数据冲突自动解决 | ⏳ |
| **自动备份** | 云端自动备份和版本回溯 | ⏳ |
